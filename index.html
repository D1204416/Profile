關於我

我是一位從醫療領域轉職至資訊工程的開發者，擁有逢甲大學資訊工程學士後學位（2024-2025）以及國立臺北護理健康大學護理系學士學位（2010-2014）。 我的職業生涯始於醫療領域，曾在衛生福利部雙和醫院擔任神經內科病房護理師，並在中山醫學大學附設醫院兒童部擔任研究助理。這些經驗培養了我細心、耐心以及解決問題的能力， 同時也讓我意識到科技對於提升醫療服務的重要性。

在過去幾年中，我對促進醫療服務的數位轉型產生了濃厚的興趣。這種興趣驅使我進一步學習資訊工程，系統式掌握軟體開發生命週期（SDLC），從瀑布模型到敏捷模型，深入理解它們的流程與應用場景。在實際專案中，我參與了開發的各個階段，學習並落實每個階段的關鍵步驟，包括需求分析、設計、實作、測試與部署。在開發過程中，我全面學習了前端與後端技術，並掌握 Vue.js 和 SpringBoot 等開發工具，資料庫 (MariaDB) 設計與管理能力，以及單元測試（JUnit）、整合測試及前端的功能測試，能有效提升系統的穩定性與可靠性。並使用 Git 進行版本控制，確保團隊協作的效率與可靠性。

除了程式開發技能，我也積極學習 UI/UX 設計，深刻理解使用者體驗在軟體開發中的重要性。為了提升這方面的能力，我在學程之外自學了 Figma、Whimsical 等設計工具，並應用於專案中進行介面設計與使用者流程規劃。力求讓產品在視覺和使用上直觀，為使用者提供高品質的操作體驗。

求職動機

我希望能夠持續運用這些技能，助力醫療相關專案的創新與發展。這些經歷不僅增強了我在軟體開發上的實力，也讓我更加深刻地體會到數位轉型對醫療服務的價值與影響。

通過軟體設計與技術實踐，我希望能實現以人為本的智慧醫療解決方案，縮短病患與醫療服務之間的距離，並為打造更高效、更人性化的醫療環境貢獻自己的力量。
